Simple console app for self-organization.

## Features
- practical console interface
- convenient task management tools
- support for windows/linux

## Coming soon
- new commands

## For Windows
Compiled app archive located in windows folder

### Installation
- Download archive
- Unzip/unrar to some folder
- To use as console command - add path to Planum to your Path variable

### Usage
To use - launch Planum.exe or type in Planum, "help" command should get you started with the app, for futher help, check project wiki

## For Linux
Compiled app archive located in windows folder

### Installation
- Download archive
- Unzip/unrar to some folder

### Usage
To use:
```
sudo ./Planum
```
from app folder or add path to folder to PATH to use as command
